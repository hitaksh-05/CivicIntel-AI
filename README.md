# CivicIntel AI

An AI-powered Smart City Incident Management Platform built for the **Vibe2Ship Hackathon**.

CivicIntel AI enables city authorities to report, monitor, prioritize, analyze, and resolve civic infrastructure incidents through a unified intelligent dashboard. The platform combines AI-assisted incident analysis, live mapping, command center workflows, predictive insights, and real-time analytics to improve municipal response and decision-making.

---

## Features

- AI-powered incident reporting
- Live city infrastructure map
- Dynamic dashboard with real-time statistics
- Incident Command Center workflow
- Resolution Tracker
- AI Analysis for incident assessment
- Predictive Insights dashboard
- AI Operations Copilot
- Real-time incident lifecycle management
- Department-wise incident monitoring
- Severity-based prioritization

## Project Workflow

1. Report a civic incident.
2. AI analyzes and classifies the incident.
3. Incident appears on the live city map.
4. Command Center assigns and advances incident status.
5. Resolution Tracker monitors progress.
6. Dashboard and analytics update automatically.
7. AI Copilot provides insights based on current incident data.

## Getting Started

### Prerequisites

- Node.js (v18 or above)
- npm

### Installation

```bash
git clone https://github.com/hitaksh-05/CivicIntel-AI.git
cd CivicIntel-AI
npm install
```

### Environment Variables

Create a `.env.local` file in the project root and add:

```env
GEMINI_API_KEY=YOUR_GEMINI_API_KEY
```

### Run the Application

```bash
npm run dev
```

The application will start locally at:

```
http://localhost:3000
```
